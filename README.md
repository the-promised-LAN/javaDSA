# javaDSA

A small library of common data structures and algorithms implemented from scratch in Java. Used for training and practice. Includes:

## Algorithms
### Sorting
* QuickSort
* MergeSort
* InsertionSort
* SelectionSort
* RadixSort
* BubbleSort
* HeapSort

### Permutation
* Manacher's Longest Permutation Algorithm

## Data Structures
* LinkedList
* Heap (Min + Max)
* Binary Search Tree
* Graph
